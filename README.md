#koa-decorate 

## 简介:
    * vue2 + koa2 + mysql 开发前后端项目
  
## 环境:
    * vue  v2.9.1
    * node v8.2.1
    * npm  v5.3.0
    
## 安装方法:
    * git clone xxxx;
    * cd frontend && npm i;
    * npm run dev;
    * localhost:8081;
    * cd backend && npm i;   
    * npm run dev;
    * localhost:3000;
