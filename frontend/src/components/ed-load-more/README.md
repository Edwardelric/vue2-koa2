# EdLoadMore

* 下拉加载更多 组件

 
## 可选的配置项：


 配置项 | 类型 | 描述 | 默认值 
------------------|---------|-------------------------------- |-----
 finished         | Boolean | 是否无数据           | false 
 immediateCheck   | Boolean | 是否在初始化时立即执行滚动位置检查    | false
 distance         | Number  | 距离底部阀值 | 10
 loadingDescClassName    | String  | 加载中样式名 | ''
 loadingNoDataClassName  | String  | 暂无数据样式名 | ''
